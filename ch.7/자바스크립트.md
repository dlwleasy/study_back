## 5.논리연산자 
참 거짓 판단해주는 연산자

(1) 같은 것을 비교하는 경우

===: 값과 타입이 모두 같아야 true -> 자주 사용권장 
==: 값만 같으면 true (타입 자동 변환)

(2) 다른 것을 비교하는 경우 

!==는 타입을 변환하지 않고 엄격하게 비교 (<-> === ) -> 자주 사용권장 
!=는 타입을 자동 변환해서 비교 (<-> == )
